# 敌人灵感 - 脑暴笔记

## 设计目标
- 保持敌方攻击可读性高，尽可能提供清晰预警和 1 回合前摇。
- 让机制与棋盘玩法联动（锁格、倒计时、危险物生成、连消惩罚）。
- 鼓励玩家做出不同决策：爆发、净化、拖回合或控盘。
- 每个敌人保持“简单循环 + 一个变招”，让战斗易学但不单调。

## 遭遇角色定位
- 重击型：节奏慢，但单体伤害重。
- 控制型：干扰棋盘状态和战斗节奏。
- 惩罚型：根据玩家行为触发反制（大连击、同色刷屏等）。
- 召唤型：制造小怪或危险物件。
- Boss 核心型：使用分阶段机制，重组已有玩法。

## 敌人概念

### 1) 荆刺陆龟（重击型）
主题：装甲野兽，专治无脑爆发。
- 普攻：龟壳猛砸（中等伤害）。
- 技能：尖刺反震（记录本回合受到的伤害，下回合返还一部分）。
- 技能：缩壳（获得护甲，1 回合内减伤）。
- 预警：反震前龟壳泛红。
- 对策：延后爆发，或先破甲再打高连击。

### 2) 灰烬小鬼（控制型）
主题：纵火系捣蛋鬼，点燃棋盘。
- 普攻：余烬弹（轻伤害 + 灼烧）。
- 技能：点燃 2 个随机格（若被消除，则对玩家队伍造成反噬伤害）。
- 技能：野火蔓延（敌方回合后，已点燃格向相邻格扩散）。
- 预警：受影响格显示燃烧符文。
- 对策：优先用小连消清掉点燃格，防止扩散。

### 3) 沼泽水蛭（续航/惩罚型）
主题：吸血寄生体，专吃被减益目标。
- 普攻：虹吸撕咬（造成伤害并按比例回血）。
- 技能：腐烂印记（目标受到的所有吸取类伤害 +X%）。
- 技能：分裂汲取（若目标生命低于 50%，本回合攻击两次）。
- 预警：被标记者出现绿色光环。
- 对策：及时净化印记，保护残血队友，优先集火水蛭。

### 4) 玻璃幽灵（节奏扰乱型）
主题：脆皮灵体，扰乱行动节奏。
- 普攻：棱镜弹（低伤害）。
- 技能：延迟印（使一名玩家技能冷却 +1）。
- 技能：碎裂（若本回合被 4 连及以上消除击杀，则触发高额反伤）。
- 被动：生命值极低。
- 对策：先慢慢压血，再用小连消或技能收尾。

### 5) 铁甲鼹鼠（棋盘控制型）
主题：掘地并堵塞棋盘空间。
- 普攻：钻击（单体伤害）。
- 技能：地道坍塌（锁定 2 个格子，持续 2 回合）。
- 技能：地震脉冲（若场上有 3 个及以上锁格，触发 AoE 脉冲）。
- 预警：锁格上显示裂地图标。
- 对策：优先解锁，或压制其锁格数量叠高。

### 6) 海妖蝠（惩罚型）
主题：声波猎手，克制“超长操作回合”。
- 普攻：尖啸（AoE 刮痧伤害）。
- 技能：回声税（本回合第 3 次消除后，每多一次消除就追加伤害）。
- 技能：夜袭俯冲（对最低生命目标造成高伤）。
- 预警：回声税激活时出现波形计量条。
- 对策：放弃贪连，选择高效率的 2-3 连消回合。

### 7) 沼地炼金师（召唤/控制型）
主题：往棋盘投掷不稳定药剂。
- 普攻：酸液瓶（伤害 + 降防）。
- 技能：生成炸弹格（倒计时 2；倒计时结束前未消除则 AoE 爆炸）。
- 技能：突变泄漏（将附近格转化为毒格）。
- 预警：炸弹格显示清晰倒计时数字。
- 对策：在敌方压力下规划路径，优先拆弹。

### 8) 霜角巨鹿（控制型）
主题：寒潮敌人，减速并冻结。
- 普攻：鹿角冲锋（伤害）。
- 技能：寒风侵袭（施加减速，降低消除获得的能量）。
- 技能：深度冻结（随机冻结一整列，持续 1 回合）。
- 终结模式：若玩家已处于减速状态，深度冻结持续时间 +1。
- 对策：尽快净化减速，避免被其建立冻结前置条件。

### 9) 遗迹守卫（防御精英）
主题：会切换架势的古代构装体。
- 姿态 A - 守御：每回合获得护盾。
- 姿态 B - 反射：反弹首个命中它的技能。
- 姿态 C - 破坏：高输出但无防御。
- 轮转：每回合按 A -> B -> C 循环。
- 对策：把爆发留在 C 姿态，用低价值技能骗掉反射。

### 10) 风暴九头蛇（Boss）
主题：多头元素 Boss，分阶段加压。
- 阶段 1（100%-70%）：每回合 1 个头行动（火/冰/雷随机）。
- 阶段 2（70%-35%）：每回合 2 个头行动，并引入危险格机制。
- 阶段 3（35%-0%）：全部头颅开始蓄力“天灾湮灭”（3 回合团灭），除非玩家清掉 3 个危险格。
- 头颅效果：
  - 火：灼烧 + 点燃格子。
  - 冰：冻结随机格簇。
  - 雷：基于当前连击数造成连锁伤害。
- 对策：第 3 阶段从纯 DPS 思路切换为“目标清除优先”。

## 攻击机制库（可混搭）
- 倒计时格：必须在计时结束前消除。
- 锁定格：无法移动/消除，需相邻消除命中或用净化技能解除。
- 腐化格：直接消除会扣资源，先净化再消除更优。
- 伏击：若敌人在本回合未受击就行动，获得额外效果。
- 反击层数：受击时叠层，下一次攻击消耗层数强化。
- 阈值护盾：仅能被 4 连及以上消除或技能伤害击破。
- 连击惩罚：玩家连击超过阈值后，敌方伤害按连击长度增长。
- 阵型链接：敌人共享生命或增益，直到“链接节点”被破坏。

## 预警设计准则
- 每类机制使用统一视觉语言：
  - 红色 = 即将到来的伤害
  - 蓝色 = 控制/冻结
  - 绿色 = 中毒/吸取
  - 黄色 = 倒计时/目标机制
- 高危险攻击至少满足以下之一：
  - 回合计数器
  - 敌人头顶图标
  - 棋盘受影响区域高亮
- 关键机制前增加短提示文本（如“天灾蓄力中”“炸弹已激活”）。

## 按关卡的强度成长思路
- 前期：每个敌人只带 1 个机制，重叠度低。
- 中期：双定位敌人增多（伤害 + 控盘），加入轻度惩罚机制。
- 后期：使用角色协同小队（控制 + 惩罚 + 重击）。
- Boss：每个阶段都加入目标打断，避免纯 DPS 竞速。

## 快速遭遇组合种子
- 种子 A：荆刺陆龟 + 灰烬小鬼
  - 问题点：爆发会被反震和燃烧扩散双重惩罚。
- 种子 B：铁甲鼹鼠 + 海妖蝠
  - 问题点：棋盘锁定叠加连击税。
- 种子 C：沼地炼金师 + 沼泽水蛭
  - 问题点：拆弹路线压力下，水蛭持续吸血。
- 种子 D（精英）：遗迹守卫 + 玻璃幽灵
  - 问题点：需要精准把握反射与碎裂的击杀窗口。

## 可选 Meta 钩子
- 地区词缀可偏向特定机制（火山 = 点燃，苔原 = 冻结）。
- 角色属性亲和可硬克某些控制类型。
- 精英修饰词可从机制库额外添加 1 条机制，提高重玩变化。
