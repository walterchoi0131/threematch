# 角色設定集

## 世界觀

天地由五大元素構成——**水、火、草、光、暗**。然而數百年前，光之國王以暗元素為不祥之力，施展禁術封印了所有暗元素的自然生成，使世人逐漸遺忘第五元素的存在，只知水、火、草、光四象。

元素學院是培育元素使者的最高學府，學生依天賦操控對應元素，透過消除同色寶石來引導元素之力進行戰鬥。

### 元素相剋

- 火（RED）→ 草（GREEN）：焚燒
- 草（GREEN）→ 水（BLUE）：吸收
- 水（BLUE）→ 火（RED）：澆熄
- 光（YELLOW）與暗（DARK）互剋

## 主要角色

### Fox（火屬主角）

| 項目 | 內容 |
|------|------|
| 資源檔 | `characters/char_fox.tres` |
| 元素 | 火（gem_type: RED / 0） |
| 種族 | 狐狸 |
| 性格 | 熱情衝動、重情重義、永不放棄 |
| 定位 | 攻擊型輸出 |

**外貌描述**：紅棕色毛皮的年輕狐狸，眼中總帶著不服輸的火焰。穿著學院制服時袖子總是捲起，露出手臂上的火焰紋路。五年後蓄了短鬚，眼神更加堅定。

**背景故事**：元素學院的問題學生，成績平平但戰鬥直覺出色。在黑暗入侵事件中目睹老師 Husky 的犧牲，立誓要找回老師。宿舍大火中帶領同伴逃生。五年後成為遊歷各地的冒險者，得知老師可能未死的消息後，決意重新集結夥伴前往魔王城。

**成長弧線**：衝動少年 → 經歷失去 → 獨自磨練 → 成為領袖 → 以信任凝聚夥伴

**技能對照**：
- **主動技能「爆炸」**：由上而下逐列引爆，CD: 1 回合
- **回應技能「Fireball」**：消除 9+ 火寶石時，生成火球高階寶石
- **回應技能「Fire Pillar」**：消除 4+ 火寶石連線時，生成火柱高階寶石

### Boar（水屬主角）

| 項目 | 內容 |
|------|------|
| 資源檔 | `characters/char_boar.tres` |
| 元素 | 水（gem_type: BLUE / 1） |
| 種族 | 野豬 |
| 性格 | 沉穩冷靜、理性至上、外冷內熱 |
| 定位 | 坦克型 / 自回復 |

**外貌描述**：深藍色鬃毛的壯碩野豬，獠牙上刻有水紋符文。學生時期溫和內斂，五年後因長期狩獵魔物而渾身傷痕，眼神陰沉疲憊，全身散發冰冷水氣。

**背景故事**：學院時期的優等生，與 Fox 和 Raccoon 是室友。黑暗入侵後獨自踏上狩獵魔物之路，親眼見證太多村莊被毀、太多人死去，逐漸對世界絕望並黑化。以極端效率獵殺一切魔物，不再相信希望。需要被 Fox 在戰鬥中打敗才能重新振作。

**成長弧線**：溫和優等生 → 目睹絕望 → 黑化獨行 → 被老友喚醒 → 回歸隊伍（仍保持冷靜性格，但重拾信念）

**技能對照**：
- **被動技能「Drinking」**：消除水寶石時，額外回復攻擊傷害 50% 的生命值
- **主動技能「Attack Form」**：將盤面所有火寶石轉化為水寶石，CD: 5 回合

| 項目 | 內容 |
|------|------|
| 資源檔 | `characters/char_raccoon.tres` |

| 種族 | 浣熊 |
| 性格 | 善良膽小、心思細膩、療癒系 |
| 定位 | 輔助型 / 寶石轉換 |

**外貌描述**：灰綠色毛皮的瘦小浣熊，戴著圓框眼鏡，總是抱著一本草藥圖鑑。五年後長高了些，但仍是三人中最矮的，身上纏著藤蔓飾品。

**背景故事**：學院的植物學天才，戰鬥能力是三人中最弱的。黑暗入侵後逃到森林深處隱居，靠草元素與植物共生勉強存活。五年後被魔物發現藏身處，正被圍攻時被 Fox 和 Boar 救下。雖然膽小，但為了保護朋友能爆發出驚人的決心。

**成長弧線**：膽小書蟲 → 獨自求生 → 被魔物欺凌 → 獲救後重新振作 → 以輔助之力守護同伴

**技能對照**：
- **被動技能「Photosynthesis2」**：每回合開始時，將 3 個寶石轉為葉寶石（優先轉化火 > 水）
- **回應技能「Leaf Storm」**：消除 6+ 葉寶石時，額外將 3 個寶石轉為葉寶石

### Crow（暗屬第四主角）

| 項目 | 內容 |
|------|------|
| 資源檔 | 待新增 |
| 元素 | 暗（gem_type: DARK / 待定） |
| 種族 | 烏鴉 |
| 性格 | 沉默寡言、警覺多疑、內心善良 |
| 定位 | 刺客型 / 暗屬輸出 |

**外貌描述**：漆黑羽毛的人形烏鴉，身披破舊的黑色斗篷，面容隱在兜帽陰影中。左眼有一道舊傷疤，紫色瞳孔在黑暗中微微發光。體型纖瘦但動作極其敏捷。

**背景故事**：身世不明的暗元素使者。在暗元素被禁封的時代，Crow 是極少數天生能操控暗元素的存在，因此從小被視為異類和災星，四處流浪。被魔物攻擊時被 Fox 和 Raccoon 救下，但遭到 Boar 的極度不信任（因暗元素與魔王軍同源）。在 Fox 和 Raccoon 瀕死時挺身而出，以暗元素之力擊退魔物群，從此加入小隊。

**成長弧線**：孤獨流浪者 → 被救卻不被信任 → 關鍵時刻證明自己 → 找到歸屬 → 揭露暗元素的真相

**技能設計方向**：
- **被動技能**：擊殺敵人時回復少量 HP／獲得暗寶石增益
- **主動技能**：將隨機寶石轉為暗寶石，或短時間內提升暗寶石傷害倍率
- **回應技能**：消除大量暗寶石時，對全體敵人造成穿透傷害

| 項目 | 內容 |
|------|------|
| 資源檔 | `characters/char_husky.tres` |
| 元素 | 光（gem_type: YELLOW / 6） |
| 種族 | 哈士奇 |
| 性格 | 溫柔嚴厲並存、有耐心、捨己為人 |
| 定位 | 導師 / 光屬輸出+治療 |

**外貌描述**：銀白毛皮的高大哈士奇，穿著學院教師袍，金色眼瞳溫和而堅定。戰鬥時全身散發金色光芒，手持光之法杖。

**背景故事**：元素學院最年輕的教師，四元素理論的研究者。負責教導 Fox、Boar、Raccoon 的戰鬥訓練。在黑暗入侵時獨自斷後，拼盡全力抵擋暗之力量，最終被神秘人吸收力量後倒下。但其實並未完全死亡——靈魂被囚禁在魔王城深處。

**成長弧線**：慈愛導師 → 英勇犧牲 → 靈魂被囚 → 最終被學生們救回

**技能對照**：
- **主動技能「There shall be light」**：進入選取模式，選擇位置將十字範圍寶石轉為光寶石，CD: 3 回合
- **回應技能「Justice Slash」**：消除 9+ 光寶石時，生成聖十字高階寶石，爆炸時造成全體敵人「寶石數 × 50 ATK」傷害並回復 20% HP

## 關鍵 NPC / BOSS

### 王女（暗元素公主）

| 項目 | 內容 |
|------|------|
| 資源檔 | 待新增 |
| 元素 | 暗（DARK） |
| 種族 | 待定（光之國王族） |
| 性格 | 堅忍不拔、憤世嫉俗、渴望被理解 |
| 定位 | 可操作角色（終章加入） / 暗屬輔助+控制 |

**外貌描述**：蒼白膚色的少女，長髮半黑半白，身穿被撕裂的王族禮服。眼中映著紫色暗焰，指尖常纏繞暗元素的黑霧。儘管衣衫襤褸，舉止間仍帶有王族的優雅。

**背景故事**：光之國王的女兒，天生便能操控被禁封的暗元素。國王視她為王國之恥，自幼將她封禁在王宮地下室，不見天日。年復一年的孤獨和黑暗中，暗元素逐漸腐化她的心智。她最終掌握了暗元素的力量，反叛並逃出王宮。國王為了追捕她，解除了暗元素的封印——這才是大地被黑暗籠罩、魔物橫行的真正原因。世人卻誤以為一切都是王女帶來的災禍。

**成長弧線**：被囚禁的孩子 → 被暗元素腐化 → 反叛逃亡 → 被世界追捕 → 真相大白後與主角們並肩作戰

**技能設計方向**：
- **被動技能**：每回合將少量寶石轉為暗寶石（與 Raccoon 的光合作用對應）
- **主動技能**：選取區域，暫時「封印」該區域寶石使其無法被消除，數回合後全部爆破造成大量傷害
- **回應技能**：消除暗寶石時有機率使敵人陷入「黑暗」狀態，降低攻擊力

| 項目 | 內容 |
|------|------|
| 資源檔 | 待新增 |
| 元素 | 五屬全能（水/火/草/光/暗自由切換） |

**外貌描述**：身穿金白鎧甲的威嚴君主，王冠上鑲嵌五色寶石，分別對應五大元素。在切換屬性時，鎧甲和王冠會改變光芒顏色。越到戰鬥後期，外表越扭曲，五色光芒開始不穩定地交錯。

**背景故事**：光之國的統治者，當年以「維護秩序」之名封禁暗元素，實則恐懼無法控制的力量。為了追捕逃跑的女兒，不惜釋放全部暗元素——他自己才是一切災禍的源頭。在終章以五屬切換的能力與主角們決戰。

**戰鬥設計**：
- 五色盤面（allowed_types: RED, BLUE, GREEN, YELLOW, DARK）
- BOSS 每隔數回合切換自身屬性，改變弱點和抗性
- 每次切換屬性時會施放對應元素的全場技能
- 多階段戰鬥：純光 → 雙屬 → 五屬全開

## 角色關係

```
        ┌─── 師生 ───┐
      Husky（老師）     │
     ╱    │    ╲       │
  Fox   Boar  Raccoon  │
  （火） （水）  （草）  │
    │     │      │     │
    └──室友/同窗──┘     │
          │             │
     五年後重逢          │
    Fox ←→ Boar         │
    （打敗黑化）         │
          │             │
        Crow（暗）       │
    ╱ 被Fox救 ╲         │
  Fox信任  Boar不信任    │
    → 證明自己後加入     │
                        │
      ┌── 父女 ──┐     │
    國王 ←敵對→ 王女    │
   （BOSS）    （暗）   │
      │               │
    神秘人──吸收力量──Husky
```

- **Fox ↔ Boar**：最好的朋友，也是最大的衝突——Boar 的黑化是 Fox 必須跨越的障礙
- **Fox ↔ Raccoon**：Fox 是 Raccoon 的保護者，Raccoon 的善良是 Fox 堅持的理由
- **Boar ↔ Crow**：最大的不信任關係，Boar 因長年與暗屬魔物戰鬥而對暗元素極度敵視
- **Fox ↔ Crow**：Fox 不顧一切救了 Crow，建立了信任的起點
- **國王 → 王女**：扭曲的父女關係，國王的恐懼造成了一切悲劇
- **Husky → 全體主角**：精神支柱，被囚禁的老師是眾人踏上旅程的動力
